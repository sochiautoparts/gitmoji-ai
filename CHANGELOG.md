# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### ✨ Features

- AI-powered commit message generation using OpenAI GPT models
- Support for Conventional Commits, emoji-style, and plain formats
- Multi-language commit messages (EN, RU, ES, DE, FR, JA, ZH)
- AI-powered changelog generation with smart grouping
- GitHub Action for CI/CD integration
- Git hook for automatic commit suggestions
- Usage tracking with free tier limits (50 commits/month)
- Pro license activation system
- Rich CLI interface with interactive selection
- Fallback mode when no API key is configured
- Diff analysis with automatic change type detection

### 📚 Documentation

- Comprehensive README with examples
- GitHub Action usage guide
- Multi-language examples

### 🧪 Tests

- Core diff analysis tests
- Commit parsing tests
- Changelog grouping tests
- Fallback generation tests
